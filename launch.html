Authentication Method

💡 How to get your token:
  1. In Overleaf: Menu → Git
  2. Click "Generate token" (or use existing one)
  3. Copy the token and paste it above
View detailed guide →

How it works

  1. Click "Launch Codespace" to open GitHub Codespaces
  2. Wait for the Codespace to start (2-3 minutes first time)
  3. Copy and run the setup command shown after clicking Launch
  4. Your repository will be cloned and configured automatically

⏱️ Note: First-time setup takes ~2 minutes to download TeXLive packages. Subsequent launches will be much faster.
